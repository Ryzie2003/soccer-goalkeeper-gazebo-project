# ROS 2 Soccer Goalkeeper

A ROS 2 Lyrical and Gazebo simulation of a mobile goalkeeper that tracks a
rolling ball, predicts its goal-line intersection, and moves to intercept it.

## Status

Project setup is in progress. The first milestone is a manually controlled
TurtleBot3 constrained to move along a simulated goal line.

## Planned MVP

- Compact Gazebo soccer goal and rolling ball
- Repeatable shot launcher
- Ground-truth ball tracking
- Linear goal-line intersection prediction
- Closed-loop goalkeeper control
- Goal/save detection and match statistics

## Workspace

Source packages live in `src/`. Generated `build/`, `install/`, and `log/`
directories are intentionally excluded from Git.
